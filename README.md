### 📠 AS
---
⚙️ 개발 환경 : eclipse, tomcat, github <br>
💻 개발 언어 : oracle, java <br>
📜 프로젝트 설명 : <br> 
전자기기 AS(애프터서비스) 예약을 위한 웹 기반 예약 관리 시스템을 개발하였다. <br>
사용자 역할을 일반 회원과 관리자로 구분하여, 보안성과 관리 효율성을 동시에 고려한 구조로 설계하였다.

일반 회원은 로그인 및 회원가입을 통해 시스템에 접근하며, 전자기기 종류와 방문 날짜·시간을 선택하여 AS 예약을 등록할 수 있다. <br>
또한 본인이 등록한 예약에 대해 조회, 수정, 삭제 기능을 수행할 수 있도록 구현하였다.

관리자는 전체 회원의 예약 정보를 통합적으로 관리할 수 있으며, 예약 내역 조회는 물론 필요 시 수정 및 삭제 기능을 통해 효율적인 예약 운영이 가능하도록 구성하였다. <br>
